# Rights and appeal path

This tool is a **research aid, not a compliance ruling.** It does not make
binding eligibility or compliance determinations, and no decision it produces is
authoritative.

## How to understand an answer
Every answer shows the exact bylaw cite and the verbatim rule text it relied on,
so users can read the underlying rule directly. If an answer cites no bylaw, the
tool reports that the question is not clearly covered rather than guessing.

## How to contest or get human review
For any rules question that affects a real decision, or if an answer seems wrong
or incomplete, contact the **UC San Diego Athletics compliance office** before
acting. The compliance office is the authoritative human reviewer and the
override path for anything this tool returns.

> Add the compliance office contact (name, email, phone) here before launch.

## Override
The tool never blocks or compels an action. Its output is advisory; the user and
the compliance office retain full judgment.
